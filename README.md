# Install

`$ npm install -g gulp`

## 清理

在开发过程中，发生产以前这个功能非常有必要

1. 需要安装del模块
2. 建立task`cleanup`
3. 清理css的task `cleanupcss`
4. 清理js的task `cleanupjs` 

## 处理javascript

1. 需要安装模块gulp-uglify, gulp-concat, gulp-rename

## postcss
1. 需要安装模块gulp-postcss, gulp-concat, gulp-minify-css
